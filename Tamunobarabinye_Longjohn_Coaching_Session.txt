In your code, I see you used <%- include(...) %> for the partials at the top and bottom, but <%= title %> for the heading in the middle. Why do we use the dash <%- for the includes instead of the equals sign <%=?

[Student]: The difference is in how EJS handles the output. <%= %> is for Escaped Output, so it prints content while escaping HTML characters for safety. <%- %> is for Unescaped Output, the include() function returns actual HTML content from another EJS file. If I used <%= %> instead, the HTML tags from the partial would be escaped and shown as plain text on the webpage instead of being rendered.

[Coach]: Exactly! If you escaped the partial, you'd see literal <html> and <nav> tags printed as text on your screen, which would definitely break the layout.
